HTML Basics

HTML stands for HyperText Markup Language.

It is used to create and structure sections, paragraphs, and links on web pages.

HTML elements form the building blocks of every website.

Common tags include <div>, <p>, and <a>.

HTML5 introduced semantic tags such as <header> and <footer>.

<!DOCTYPE html> <html> <head></head> <body></body> <p>Hello World</p>
  • Use semantic tags for better accessibility.
  • Close all HTML tags properly.
  • Use attributes to add more information to elements.
  • Use forms to collect user input.
  • Include images with <img> tags.
CSS Basics

CSS stands for Cascading Style Sheets.

It is used to style HTML elements, controlling colors, fonts, and layout.

CSS rules consist of selectors and declarations.

Use classes and IDs to target elements effectively.

Modern layouts can be created using Flexbox and Grid.

p { color: blue; } .container { display: flex; } #header { font-size: 24px; } ul li { margin-bottom: 10px; } @media (max-width: 768px) { ... }
  • Use external stylesheets for maintainability.
  • Use semantic selectors for better readability.
  • Box model includes margin, border, padding, and content.
  • Media queries make your site responsive.
  • CSS transitions and animations can add interactivity.
JavaScript Basics

JavaScript is a programming language for the web.

It allows dynamic content and interactivity.

Variables store data using let or const.

  • Functions allow code reuse.
  • Conditionals control logic with if statements.
  • Loops iterate over data.

Events let you respond to user actions.

DOM manipulation is done via JavaScript.

Arrays and objects store collections of data.

JavaScript ES6+ adds features like arrow functions and template literals.

Always test your code in multiple browsers.

DOM Manipulation

The DOM (Document Object Model) represents the structure of a web page.

You can select elements using methods like document.querySelector() or document.getElementById().

Content can be changed dynamically with innerHTML or textContent.

Styles can be modified using the style property.

Elements can be added or removed with appendChild() and removeChild().

const heading = document.getElementById("title"); heading.textContent = "Hello World"; const newDiv = document.createElement("div"); document.body.appendChild(newDiv); element.classList.add("active");
  • Select elements using IDs, classes, or tags.
  • Change content dynamically.
  • Modify styles on the fly.
  • Add or remove DOM elements as needed.
  • Use event listeners to react to user interactions.
APIs and Fetch

APIs allow different software applications to communicate with each other.

The Fetch API is used to make network requests in JavaScript.

Responses from APIs are often in JSON format.

Asynchronous code can be handled using Promises or async/await.

Always handle errors properly to avoid breaking the application.

fetch("https://api.example.com/data") .then(response => response.json()) .then(data => console.log(data)) async function getData() { const response = await fetch(url); } try { ... } catch(error) { console.error(error); }
  • Use fetch() to request data from servers.
  • Handle asynchronous operations with Promises.
  • Use async/await for cleaner code.
  • Check response status before using the data.
  • Document API calls for maintainability.
